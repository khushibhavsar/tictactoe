*What is Tic Tac Toe?*
Tic Tac Toe is a simple game involving two compoenents "x" and "o". The first player to have these components three in a row is the winner!

*Why was this project made?* 
This project was made to learn the basics of javascript. Through the help of different functions and variables the game taught me how such aspects can be used. 

*How was this project made?* 
This project was made with HTML, CSS, and Javascript. I used different articles from geeksforgeeks.org and youtube videos to learn about the variables and how this game is made. I learned a lot about arrays and how they can be used to store values, perfect for this game. 

*What you struggled with + what I learned.*
This project was an initial stuggle, as being a beginner coder it was my first time being in contact with if and else statements, boolean, variables, and other types of functionalities. I learned a lot about the ways you can simplify your code to make it look pretty. As well as different extensions that organize your code for you. Also had a really fun time in being able to make the website look aesthetically pleasing and testing the different layout options with the help of css.
